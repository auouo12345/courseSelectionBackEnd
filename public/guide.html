選課系統說明書

歡迎使用放假大學選課系統，本說明旨在協助大學學生順利完成選課流程，包括註冊登入、課表查詢、課程加選、退選、以及選課通知等功能。請仔細閱讀以下說明，以確保能充分利用選課系統的所有功能。

1.使用者註冊與登入

學生需使用學校的電子郵件註冊並登入系統。登入後，您將能根據不同的身分（學生或教師）進行相應的選課操作。

登入流程：

  • 選擇您的身分（學生或教師）。
  • 輸入學校信箱及密碼，然後點擊登入。
  • 若登入成功，將自動導向選課頁面；若密碼錯誤，系統將顯示提示。

2. 課表查詢

登入系統後，學生可以查看當前學期的完整課表。課表中包含所有已加選的課程、關注列表，以及預選課程。

查詢方式：

  • 登入後，系統會自動顯示當前學期的課表。
  • 課表中列出所有已匯入的課程，包括時間、地點等詳細信息。

3. 加選課程

學生可以在課程檢索頁面瀏覽課程並進行加選。系統會自動檢查課表是否衝突、課程餘額等條件，以確保選課成功。

加選流程：

  • 在課程檢索頁面中瀏覽並選擇想加選的課程。
  • 系統會檢查課程衝突、身分類別等條件。
  • 若符合所有條件，課程將成功加選並加入學生的課表。
  • 若不符合條件，系統會顯示錯誤提示（例如：衝堂、課程已滿等）。

4. 退選課程

學生可以退選已加選的課程。退選後，系統會自動更新學生的課表及課程的選課人數。

退選流程：

  • 查看已加選的課程列表，選擇要退選的課程，然後點擊退選按鈕。
  • 系統會顯示確認對話框，詢問是否確定退選。
  • 若確認退選，該課程將從課表中移除，並更新課程的餘額。

5. 選課通知與提醒

在選課期間，系統會提供各類通知，包括選課截止日提醒及課程變動，幫助學生掌握重要資訊。

通知方式：

  • 登入系統後，系統會顯示選課及退選的截止日期。
  • 當課程時間、地點或授課教師發生變動時，系統將自動發送通知。

6. 課程檢索

學生可以使用關鍵字或篩選條件來搜尋可選課程，並查看詳細資訊，例如課程內容、評分標準、課程教材等。

檢索方式：

  • 在課程檢索欄中輸入關鍵字（例如課程名稱、授課教師、課程類別等）。
  • 系統根據檢索條件返回符合的課程列表，點擊任一課程可查看其詳細內容。

7. 使用者注意事項

  • 每學期的選課期間，系統可以承受大量學生同時操作，請放心使用。
  • 請注意選課及退選的時間限制，避免錯過重要日期。
  • 選課時請留意課程衝突提醒及課程餘額，確保選課順利。
  • 系統兼容多種主流瀏覽器，建議使用 Chrome、Firefox 或 Microsoft Edge 獲得最佳體驗。

我們致力於提供簡單明瞭的選課介面，幫助學生輕鬆管理課程。如有任何問題，請隨時聯繫資訊處尋求幫助。祝您選課順利！
